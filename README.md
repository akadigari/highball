# vane

The weather desk. Predicts tomorrow's official high temperature for the
cities Kalshi runs daily markets on, turns those predictions into sim
bets with a clear entry time and exit rule, and keeps receipts.

Sim only. No real orders, no real dollars, ever.

Spec first: read [SPEC.md](SPEC.md). The lab gets built to that
document. Gates are written down before results exist, so a fail is a
finding, not an embarrassment.
